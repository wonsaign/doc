
Sublime Text 3 超详细插件安装

1. [command] + [shift] + P 打开管理器
2. 输入 -> install package control
3. 安装插件
   1. ChineseLocalizations中文汉化
4. 查看已经安装的软件
   1. [command] + [shift] + P 打开管理器
   2. 选择“Package Control：List Packages”
5. 删除已经安装的插件
   1. [command] + [shift] + P 打开管理器
   2. 选择“Package Control：Remove Packages”
